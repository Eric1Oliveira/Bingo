⚙️ Falta conectar o Supabase

Abra o arquivo js/config.js e cole a URL e a anon key do seu projeto Supabase. Depois rode o sql/schema.sql no SQL Editor.

  1. Crie um projeto em supabase.com
  2. Project Settings → API → copie Project URL e anon public
  3. Cole em js/config.js
  4. SQL Editor → cole sql/schema.sql → Run
  5. Recarregue esta página 🎉
